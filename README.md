# Desmos Art Generator

By Huatao

## HOW TO USE THIS SCRIPT:

- Step 1.: Open terminal.app and copy the following code:
```
python3 -m pip install pip
pip3 install opencv-python
pip3 install numpy
```

- Step 2. Get a image and drag it onto the Desktop. Name it test.jpg or change the path variable below.

- Step 3. Run this script. It will generate a text file on your Desktop named desmos_equations.txt.

- Step 4. Open desmos_equations.txt, copy all the equations, and paste them into Desmos (https://www.desmos.com/calculator).


## NOTES:

1. GRAPHING_INACCURACY_VALUE controls the accuracy of the approximation.
The lower, the more accurate but the more equations.

2. fhand is the output file where the Desmos equations will be written. Edit the USERNAME to your
macOS username.

3. path is the input image path. Edit the USERNAME to your macOS username.
